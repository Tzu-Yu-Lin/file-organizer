# File Organizer

A simple Python tool that organizes files in a folder by file type.

## Features
- Scan a folder
- Categorize files by extension
- Create folders automatically
- Move files into the correct folder
- Handle duplicate file names
- Preview mode
- Show summary after organizing

## Technologies Used
- Python
- pathlib
- shutil

## How to Run

1. Make sure Python is installed.
2. Open the terminal in the project folder.
3. Run:

```bash
python main.py
